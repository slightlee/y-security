# y-security 介绍
    这是基于 Springboot、Mybatis、Mybatis Plus、Shiro权限框架 等技术搭建的后台管理项目。
    Swagger 接口工具 、Thymeleaf 模板引擎、logback 日志组件、layui-Template

# 项目结构说明：
    
    y-security                                      -- 管理版本号
       |
       |-- y-admin                                  -- 后台管理模块            
       |-- y-generator                              -- 代码生成模块
       |-- y-common                                 -- 公共模块
